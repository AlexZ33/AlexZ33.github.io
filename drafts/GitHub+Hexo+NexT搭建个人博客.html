1. Hexo介绍

Hexo 是一个简单地、轻量地、基于Node的一个静态博客框架。通过Hexo我们可以快速创建自己的博客，仅需要几条命令就可以完成。

发布时，Hexo可以部署在自己的Node服务器上面，也可以部署github上面。对于个人用户来说，部署在github上好处颇多，不仅可以省去服务器的成本，还可以减少各种系统运维的麻烦事(系统管理、备份、网络)。所以，基于github的个人站点，正在开始流行起来….

Hexo的官方网站：http://hexo.io/ ，也是基于Github构建的网站。

4. 博客远程部署

以上预览中的网页仅仅是存在于本地，自己可以通过执行hexo server （或者hexo s）命令来预览，但是别人是看不到的，所以我们需要进一步将博客部署到远程，这里指的远程就是GitHub（或者说Github Pages）。

  • 注册GitHub

进入Github官网,点击右上角Sign up按钮，进行注册，这里没有什么特别注意的地方，一路填写相关信息就行。

  • 创建远程仓库

注册完Github账户后，登陆自己的账户，在主页面右上角，一个加号＋的旁边有个倒三角形，点击倒三角形，选择New repository新建一个仓库。创建一个用于存储博客的仓库。注意:一定注意这个特殊仓库的命名，名字一定是<username>.github.io,比如我的账户用户名是AlexZ33，因此我创建的仓库名字是AlexZ33.github.io
